# onestep_install
这是一个用于自己快速安装指令的仓库
## 01 ROS
```
wget http://fishros.com/install -O fishros && . fishros
```

## 02 一键安装src路径下所有package的依赖项

```
rosdep install --from-paths src --ignore-src --rosdistro melodic -r -y
```
